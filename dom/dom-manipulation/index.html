DOM Manipulation

The style property is used to style css on a targeted element

Separation of Concerns: It is recommended for styles to be defined in separate files. The style property allows for quick styling, for example for testing purposes.

define a class with styles in CSS, and use JS when you want to implement those properties. Ex:

In CSS Document ---> .aClass {
{color: purple;
fontSize: 76px;
}

Then, in JS Document ----->
var tag = document.querySelector('h1');

tag.classList.add('aClass');
tag.classList.remove('aClass');
tag.classList.toggle('aClass');
